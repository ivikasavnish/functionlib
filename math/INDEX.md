# MATH Functions Index

Total Functions: ~2,500+

## Segments/Groups

### 1. Algebra (350+ functions)
Linear equations, quadratic equations, polynomials, factoring, systems of equations, sequences, series, exponential and logarithmic functions.

**Sample Functions:**
- solve_linear_equation - Solves linear equations
- solve_quadratic_equation - Solves quadratics
- factor_polynomial - Factors polynomials
- solve_system_linear - Solves systems
- binomial_expansion - Expands binomials

### 2. Calculus (350+ functions)
Derivatives, integrals, limits, differential equations, optimization, related rates, multivariable calculus.

**Sample Functions:**
- derivative - Computes derivatives
- integral - Computes integrals
- limit - Calculates limits
- critical_points - Finds extrema
- differential_equation - Solves DEs

### 3. Geometry (150+ functions)
2D shapes, 3D solids, coordinate geometry, transformations, conic sections, measurement calculations.

**Sample Functions:**
- distance_2d - Distance between points
- triangle_area - Triangle area
- circle_area - Circle area
- rotate_point - Rotations
- sphere_volume - Sphere volume

### 4. Trigonometry (100+ functions)
Trig functions, inverse trig, identities, solving triangles, wave functions, circular motion.

**Sample Functions:**
- sin, cos, tan - Basic trig functions
- law_of_sines - Triangle solving
- double_angle_formula - Trig identities
- degrees_to_radians - Conversions
- unit_circle_point - Unit circle

### 5. Statistics (100+ functions)
Descriptive statistics, probability distributions, hypothesis testing, regression, correlation.

**Sample Functions:**
- mean, median, mode - Central tendency
- standard_deviation - Spread
- normal_distribution - Distributions
- t_test - Hypothesis testing
- linear_regression - Regression

### 6. Probability (100+ functions)
Combinatorics, probability distributions, Bayes theorem, stochastic processes, Markov chains.

**Sample Functions:**
- combination - Combinations
- permutation - Permutations
- bayes_theorem - Bayesian inference
- markov_chain - Markov processes
- monte_carlo_simulation - Simulations

### 7. Number Theory (200+ functions)
Prime numbers, divisibility, modular arithmetic, GCD/LCM, Diophantine equations.

### 8. Linear Algebra (200+ functions)
Vectors, matrices, determinants, eigenvalues, linear transformations, vector spaces.

### 9. Discrete Math (200+ functions)
Graph theory, combinatorics, logic, set theory, relations, Boolean algebra.

### 10. Complex Numbers (200+ functions)
Complex arithmetic, polar form, De Moivre's theorem, complex functions.

### 11. Optimization (200+ functions)
Linear programming, gradient descent, constraint optimization, convex optimization.

### 12. Numerical Analysis (200+ functions)
Root finding, interpolation, numerical integration, approximation methods.
